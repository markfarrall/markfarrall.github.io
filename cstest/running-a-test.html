Running a test

img of the app with the status area

To run a test, click the Test dropdown box and then click the Run button. Depending on the type of test you have chosen various windows may appear while the test is running. The status of the test is displayed below the list of tests. Once the test is completed the results and timings for the test are added to the recent tests section.
