Settings

Settings allow you to change the behaviour of csTest and control the tests you use. To access settings click the menu icon next to the csTest title, then click the Settings menu option.

The settings sections are:

  • Test runner - configure how the testing behaves
  • Logging - turn logging on or off and choose the level of messages logged
  • Tests - upload new tests and update or delete existing tests
  • Reset - return to the default configuration

Test Runner

In this section you can set how long the test runner should wait for the webserver to respond (in milliseconds) and which browsers should be used if the test requires use of a browser. Once you have made your changes click the Save button. To undo any changes click the Cancel button.

Logging

To set the variables used in a test, select your test from the Choose your test list. This will then list any variables used in the test and the current value used by the test. Once you have made your changes click the Save button. To undo any changes click the Cancel button.

Tests

This section allows you to:

  • Delete any tests you may have uploaded
  • Update any existing tests
  • Add new tests
  • Export a tests configuration

Delete
If you want to remove a test you have previously uploaded, select the test from the Choose your test list. Then click the Delete button.

Update Test
If you need to upload a new test file or update the test variables for a test you've previously uploaded you can use this page. Firstly choose the test you want to update from the Choose your test list. Then click the Choose Test or Choose Config buttons to choose the files to be used to update the test you have chosen, then click the Update button

New Test
To add a new test to csTest click the Choose Test or Choose Config buttons to choose the files to be used to create the test, then click the Create button

Export Config
To export the test variables for a test, firstly choose the test from the Choose your test list. Then click the Export button. Enter the name of the new file, choose the folder where it will be created and then click the Save button

Reset

Clicking the Reset button will reset csTest to the default settings. You may want to use this to remove all tests and test variables you may have added to the application.
