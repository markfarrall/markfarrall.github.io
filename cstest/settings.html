Settings

To access settings click the menu icon next to the csTest title, then click the Settings menu option.

Currently there are three groups of settings:

  • Test runner - allows you to configure the behaviour of the test tool
  • Tests - allows you to configure the values used in your tests
  • Upload test - allows you to upload a new test or a new version of an existing test to the tool

Test Runner

In this section you can set how long the test runner should wait for the webserver to respond (in milliseconds) and which browsers should be used if the test requires use of a browser. Once you have made your changes click the Save button. To undo any changes click the Cancel button.

Tests

To set the variables used in a test, select your test from the Choose your test list. This will then list any variables used in the test and the current value used by the test. Once you have made your changes click the Save button. To undo any changes click the Cancel button.

Upload Test

If you have a new or updated test use this page to add it to csTest. Click the Choose Test button to choose the file that contains the test file to be uploaded. The should generally be a file with a .js extension. The person creating your file may have also sent a configuration file along with the test file. This file should have a .json extension and will contain the default values of any test variables and other test configuration information.
