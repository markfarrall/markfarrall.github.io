Variables

The variables area allows you to define different values for your tests in different environments. For example, this allows you to define a different URL for your production or test environment. In the screenshot above different values are defined for the simple Google test. In the Italy environment the test searches for canoli using the Italian Google search page. In the Default environment, which isn't visible, the test searches for the word testy using the default Google site.

To maintain variables you can either change the values in the existing environments or you can create new environments to store new sets of information. To create a new environment you can either choose the Save As button at the bottom of an existing environment, or you can use the New tab. Either option will ask you for a name for your new environment and then create the environment. When it is created you will see it as a new tab in the dialog.

You can also delete old environments using the Delete Environment button, or rename existing ones using the Rename button
