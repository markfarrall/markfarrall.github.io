Running a test

To run a test, choose the test you wish to run from the Choose your test list. If your test contains variables you will also be shown the list of environments defined for that test. Choose the environment you want to use for the test.

If you need to change variables in your environments click the Edit Variables button. The Variables page has more information on managing these.

Once you have chosen your test and environment click the Run Test button. Depending on the type of test you are running various windows may appear while the test is running.

The results from the tests will be shown in the Test Status dialog. Once the test is completed the results and timings for the test are added to the recent tests section.If you encounter an error during the testing, or the application stops responding. You can click the Cancel Test button to choose another test, or restart your current test.
